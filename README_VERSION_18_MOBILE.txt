VÉRTIKA FIT 18 · MOBILE SIMPLE

Base:
- Versión 17 local estable sin Firebase.

Mejoras:
- Vista móvil simplificada.
- Menú inferior con botones: Inicio, Perfil, Menú, Progreso, Entrenos, Copia.
- Botones laterales para pasar de sección.
- Sidebar oculto en móvil.
- Tarjetas a una columna.
- Más legible en pantalla pequeña.
- Mantiene login local y exportar/importar copia de seguridad.
- Sin Firebase y sin errores de API.

Para probar en móvil:
1. Sube todo a GitHub sustituyendo archivos.
2. Borra caché o abre en incógnito.
3. En PC: F12 > Application > Service Workers > Unregister.
4. Ctrl + Shift + R.
