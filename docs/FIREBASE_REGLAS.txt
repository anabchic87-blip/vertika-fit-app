REGLAS FIRESTORE RECOMENDADAS PARA VÉRTIKA FIT

En Firebase → Firestore → Reglas, puedes usar estas reglas:

rules_version = '2';

service cloud.firestore {
  match /databases/{database}/documents {

    match /usuarios/{userId} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }

    match /transformaciones/{userId} {
      allow read, write: if request.auth != null && request.auth.uid == userId;
    }
  }
}

Esto permite que cada usuario solo pueda leer y escribir sus propios datos.
