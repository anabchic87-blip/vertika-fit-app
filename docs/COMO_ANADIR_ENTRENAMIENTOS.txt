CÓMO AÑADIR NUEVOS ENTRENAMIENTOS A VÉRTIKA FIT

Estructura:
- workouts/  -> aquí van las rutinas en HTML
- data/      -> datos de apoyo en JSON
- css/       -> estilos
- js/        -> funciones
- pages/     -> futuras páginas
- docs/      -> documentación

Para añadir una rutina:
1. Envías la rutina en Word o texto.
2. Se convierte a HTML con el mismo diseño VÉRTIKA.
3. Guardas el archivo en workouts/, por ejemplo:
   workouts/piernas-hyrox.html
4. En index.html se añade una tarjeta con enlace a esa rutina.
5. Subes solo el nuevo HTML y el index actualizado.

Así no tendrás que borrar ni subir todo el repositorio cada vez.
