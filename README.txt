# VÉRTIKA FIT PWA

Esta versión está preparada para instalarse en el móvil como PWA.

## Cómo probarla en ordenador
Abre `index.html` en el navegador.

## Para que aparezca “Añadir a pantalla de inicio”
Súbela a GitHub Pages o a un hosting HTTPS.

### Android / Chrome
1. Abre la web.
2. Toca los tres puntos.
3. Pulsa “Añadir a pantalla de inicio” o “Instalar app”.

### iPhone / Safari
1. Abre la web.
2. Toca compartir.
3. Pulsa “Añadir a pantalla de inicio”.
