VÉRTIKA FIT 16 · LOCAL STABLE SIN FIREBASE

Incluye:
- Login visual local, sin Firebase.
- Crear cuenta local.
- Iniciar sesión local.
- Cerrar sesión.
- Guardado local en el navegador.
- Exportar copia de seguridad JSON.
- Importar copia de seguridad JSON.
- Sin errores de API Key.
- Funciona en GitHub Pages, XAMPP, PC y móvil.

Importante:
- Los datos se guardan en el dispositivo/navegador.
- Para pasar datos a otro dispositivo, usa Copia seguridad > Exportar y luego Importar.
- Firebase queda desactivado en esta versión.

Para subir a GitHub:
1. Sube TODO el contenido sustituyendo los archivos actuales.
2. Commit changes.
3. Abre la web.
4. F12 > Application > Service Workers > Unregister.
5. Ctrl + Shift + R.
